**1. Root Cause:** 
The `grep` pattern searches for the literal string `"openclaw-gateway"` (with a hyphen), but the actual process command line separates "openclaw" and "gateway" with file paths and spaces.

**2. Minimal Corrected Line:**
```bash
CURRENT_PID=$(pgrep -f "openclaw/dist/index.js gateway" | head -1)
```
*(Note: Replaced the brittle `ps | grep | grep -v grep | awk` pipeline with the standard `pgrep -f` which ignores itself automatically.)*

**3. Test:**
```bash
pgrep -fl "openclaw/dist/index.js gateway"
```
